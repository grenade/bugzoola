open source

bugzoolla is open source software, hosted on github. it's not all that complicated really. it's built on the shoulders of giants.

  • the back end is bugzilla of course. that's a pretty complicated giant. without it, nothing here works.
    • i rely on the latest (5.1) version of bugzilla which has an almost modern, rest api.
    • by default, bugzoolla connects to the bugzilla instance at bugzilla.mozilla.org. you could probably point it at other instances of bugzilla, as long as they have the new restful api.
  • the front end is a bog standard, yeoman (generator-angular) scaffolded web ui. i used it because, it's super easy to make it look like you know what you're doing.
    • the javascript framework that ties the front end to the back, is the almighty angularjs.
    • yeoman uses twitter bootsrap to make everything look so darn pretty.
